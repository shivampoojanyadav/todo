# 📝 Todo App (Django)

Hi 👋, I'm Shivam!
This is a simple yet practical **Todo Application** built using Django to help manage daily tasks efficiently.

I built this project while learning backend development, and it helped me understand how real-world applications handle data, user interaction, and logic.

---

## 🚀 Features

* ✅ Add new tasks
* ✏️ Update/edit tasks
* ❌ Delete tasks
* 📋 View all tasks in one place
* 💾 Data stored using SQLite
* ⚡ Clean and beginner-friendly UI

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS
* **Database:** SQLite

---

## 📂 Project Structure

```bash
env/                # Virtual environment
todo_main/          # Main Django app
templates/          # HTML templates
db.sqlite3          # Database
manage.py           # Django management script
```

---

## ⚙️ How to Run Locally

Follow these steps to run the project on your system:

```bash
# Clone the repository
git clone <your-repo-link>

# Navigate into the project folder
cd ToDo

# Activate virtual environment
# (PowerShell)
.\env\Scripts\Activate

# Install dependencies
pip install django

# Run migrations
python manage.py migrate

# Start server
python manage.py runserver
```

Now open 👉 http://127.0.0.1:8000
